# 更新日志

## v1.1.6

- SH 入场再收紧：触发 K 线同根扫到止损、仅插针到达入场价或现价已在目标区域的信号会被过滤，减少“本来就不该进场”的误伤。
- SH 评分继续校准：新增周期倾向、入场距离、止损宽度、目标是否已被吃掉的调整，让分数更靠近可执行性。
- SH 复盘新增过滤统计、拒绝理由和入场确认缓冲 % 设置，方便从“进哪些”进一步优化到“为什么没进”。

## v1.1.5

- 新增 `US $100B+ Top 200` 美股池，并补充质量 / 成长 / 低估值三个子池，方便按主题进入 RSI 与 Signal Hunter。
- 管理页新增 `Clean Yahoo`，可清理当前已跟踪股票中的无效 Yahoo ticker，减少后续扫描噪音。
- 调整美股休市逻辑，手动和启动全量刷新仍可抓取股票历史 K 线，方便跨时区复盘。

## v1.1.4

- 强化 Signal Hunter：加入 OI / 资金费率参考，提高短线 OI 权重，并新增“资金配合 / 资金背离 / 费率拥挤 / OI缺失”筛选。
- 优化 SH 信号质量控制：过滤已到 TP 区域的追单信号，避免现价已经接近止盈目标后才提示。
- 重整 SH 表格体验：新增紧凑/详细模式，默认收起高级筛选区，提升可视区域和扫盘效率。
- 新增 SH 信号编号：SH 页面、SH复盘、24H战绩和交易日志统一显示可搜索编号，方便跨页面定位同一笔信号。
- 完善 SH复盘：新增 24H 战绩视图，交易日志单独归档，并在历史数据加载时自动补全信号编号。
- 调整浅色/深色模式下的表格、筛选、复盘页面可读性。

## v1.1.3

- 重新整理了 SH 复盘流程，新增交易日志页，止盈/止损记录会单独归档，方便后续回看。
- SH 复盘补充了止盈目标位，并把止盈/止损规则收紧，避免过早结算。
- 提醒页拆分为“事件中心”和“提醒设置”，保留规则配置，同时把 AI 事件单独展示。
- 管理品种页支持加密货币合约 Top50 自动筛选，减少低流动性标的干扰。
- 全局新增浅色/深色主题切换，并修正了多处深色模式下的可读性问题。
- 统一更新版本号到 v1.1.3，并重新打包发布。
