# 市场 RSI 热力图 v1.0.4

加密货币与美股多周期 RSI 监控桌面工具，基于 Electron + React 构建。

## 更新日志

### v1.0.4
- **成交额优先视图**：热力图与首页列表支持按 24h 成交额筛选 Top 50 / Top 100 / Top 200 / 全部，列表新增成交额列与排序、CSV 导出成交额字段
- **量价结构信号**：新增放量突破、放量破位、量价背离等结构信号与评分，首页可按突破 / 破位 / 背离 / 强信号过滤
- **提醒规则升级**：策略提醒支持多选、最低评分、规则预览；提醒等级改为一级 / 二级 / 三级
- **Top50 跟随提醒**：可将成交额 Top50 设置为提醒范围，自动刷新时仅在候选数量完整时同步，避免接口临时缺数据导致规则被截短
- **提醒降噪**：同一轮刷新中，同一标的、同一时间框、同一信号只提醒一次；提醒触发改为整轮市场数据刷新完成后再判断
- **数据加载稳定性**：市场数据采用分片合并、请求 ID 防串扰、自动刷新防重叠、K 线请求短重试；接口临时失败时不会用缺字段的新数据覆盖旧字段
- **规则管理重设计**：提醒页左侧品种选择栏收窄，右侧规则改为卡片列表，周期、策略、阈值、等级以标签展示
- **数据质量提示**：首页列表对成交额缺失、成交额偏低、价格缺失、K 线不足的标的显示提示
- **提醒复盘与表现统计**：提醒记录可点开查看触发价格、当前价格、触发后涨跌，并可直接打开 K 线图定位提醒点；记录栏显示可评估提醒的命中率
- **提醒模板**：提醒页新增 RSI 超卖反弹、多周期共振、熊市背离、量价突破等模板，一键填入常用规则
- **分组增强**：提醒页可直接按分组批量选择品种；品种管理页支持生成加密货币 / 美股预设组，并搜索分组成员
- **本地 K 线缓存**：OHLCV 数据缓存到本地，接口短时失败时可回退到上次缓存，减少 API 压力并改善启动/刷新体验
- **分级通知**：弹窗、声音、Webhook 可分别设置最低提醒等级，普通提醒可只记录，重要提醒再推送
- **配置备份**：设置页支持导入 / 导出配置，导出内容会自动排除 Telegram Token 与 Discord Webhook
- **更新检查**：设置页可检查 GitHub Releases，新版本发布后可从 release 页面下载

- **React Error Boundary**：全局捕获渲染异常，出错时显示友好提示与重试按钮，不再白屏

## 功能特性

### 主界面
- **RSI 热力图**：以气泡图展示当前时间框下的 RSI 分布，支持市场类型、成交额 TopN、RSI 区间与分组筛选
- **统计表格**：多周期（15m / 1h / 4h / 1d）RSI 一览，含价格、成交额、24h 涨跌幅、量价信号、评分、小火花图
- 列排序（成交额、RSI、涨跌幅、评分、量价信号）、品种搜索（优先精确匹配）、置顶、分组筛选
- RSI 变化方向箭头、背离标记（↗ 看涨 / ↘ 看跌）
- 量价信号支持 hover 查看判断说明，数据异常以提示标记展示

### 图表（双击行打开）
- K 线图 + RSI 副图，背离自动标注连线
- RSI 均线（SMA / EMA / RMA / WMA）或布林带叠加
- 多周期快速切换

### 提醒系统
- 支持 RSI 阈值、价格突破/跌破（绝对值或 % 偏离）、24h 涨跌幅超限、RSI 背离、量价结构信号
- 内置常用规则模板，支持按分组批量选择品种
- 策略提醒支持放量突破、放量破位、量价背离多选，并可设置最低评分
- **Top50 跟随**：按成交额 Top50 自动维护提醒范围，刷新数据不完整时不会截短已有规则
- **共振**：勾选后要求所选周期同时满足规则后才提醒
- 提醒等级：一级提醒 / 二级提醒 / 三级提醒；弹窗、声音、Webhook 可分别设置最低触发等级
- 批量启用 / 禁用规则
- 右下角弹窗通知（含品种名、当前值、触发阈值、实时价格、涨跌幅）
- 提醒记录（按类型 / 品种过滤），支持点击复盘触发后的价格表现并跳转 K 线图
- 推送：Telegram Bot、Discord Webhook

### 品种管理
- 币安现货 / 永续合约 / TradFi 合约一键勾选，支持全选 / 取消全选
- 美股 Ticker 验证后加入观察列表（Yahoo Finance），支持全选 / 取消全选
- 自定义分组，主界面按组筛选；提醒页可按分组批量选择品种

### 设置
- 刷新间隔、提醒冷却、RSI 参数（周期、超买/超卖阈值）
- RSI 均线类型及参数、布林带倍数
- 静默时段、开机自启、弹窗 / 音效开关
- Telegram / Discord 通知配置与测试
- 配置导入 / 导出（不会导出 Telegram / Discord 密钥）
- GitHub Releases 更新检查

## 技术栈

| 层         | 技术                                      |
|------------|-------------------------------------------|
| 桌面框架   | Electron 28                               |
| 前端       | React 18 + Vite + Zustand                 |
| 图表       | ECharts 5（热力图 + K线 + RSI）           |
| 数据源     | Binance REST API、Yahoo Finance（非官方） |
| 虚拟滚动   | @tanstack/react-virtual                   |
| 打包       | electron-builder（NSIS 安装包）           |

## 安装与使用

### 直接运行安装包

从 [Releases](https://github.com/waruiiko/RSI-inspection/releases) 下载最新 `.exe` 安装包，双击安装即可。

### 开发模式运行

```bash
# 安装依赖
npm install
cd renderer && npm install && cd ..

# 启动开发模式（热重载）
npm run dev
```

### 打包

```bash
npm run dist
# 输出到 dist/ 目录
```

## 数据说明

- **加密货币**：通过 Binance 公开 API 拉取 OHLCV 数据，无需 API Key
- **美股**：通过 Yahoo Finance 非官方接口，仅在美股交易时间内更新
- 所有数据本地计算 RSI，不上传任何数据到服务器
- K 线数据会在本地缓存，用于减少重复请求和接口临时失败时的回退显示

## 配置文件位置

设置、提醒规则、自选列表等均保存在 Electron `userData` 目录（Windows 下为 `%APPDATA%\market-rsi\`），不在项目目录中。
