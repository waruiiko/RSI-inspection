# 市场 RSI 热力图 v1.0.7

加密货币与美股多周期 RSI 监控桌面工具，基于 Electron + React 构建。适合快速查看市场 RSI 分布、量价信号、背离信号，并通过桌面提醒、Webhook 和 Codex 复盘辅助交易观察。

## 更新日志

### v1.0.7

- **AI 候选池**：新增独立 AI 页面，本地规则先筛候选，Codex 只做二次过滤和提醒归类，不提供交易指令。
- **自动 AI 筛选**：支持开启后台自动筛选，并可配置 AI 冷却时间、候选数量和启动延迟；只有候选明显变化且冷却结束时才运行 Codex。
- **AI 结果快照**：AI 页面拆分为“AI结果”和“当前候选”，AI 结果会冻结保存，不会被实时行情刷新冲掉。
- **单品种 AI 筛选**：候选列表每行支持单独运行 Codex 筛选，结果会合并回当前 AI 快照。
- **AI 提醒写入**：AI 判断为重点、风险或高置信观察的候选可写入提醒记录，并支持弹窗、声音和 Webhook。
- **Webhook AI 过滤**：Telegram / Discord 默认只发送经过 AI 筛选后的提醒，普通规则提醒保留在本地记录；设置页可切换回全部发送。

### v1.0.6

- **首屏快速启动**：启动后优先加载 4h 周期、成交额 Top 50，让窗口和首页更快进入可用状态。
- **后台分阶段补数据**：启动 2 分钟后补齐 Top 50 的 15m / 1h / 1d，启动 5 分钟后再补全全部品种和周期。
- **启动期提醒静默**：启动补数据阶段只刷新快照，不立刻触发提醒判断，避免刚打开软件时误触发或抢占资源。
- **K 线缓存优化**：市场缓存从单个大型 `market-cache.json` 改为 `market-cache-v2` 小文件按需读取，减少首次启动 JSON 解析压力。
- **指标后台线程计算**：RSI、背离、量价结构与评分计算迁移到 Worker Thread，降低主进程刷新压力。
- **提醒记录右侧面板优化**：提醒记录可占据首页右侧区域，列表可显示更多历史提醒。
- **Codex 复盘入口**：提醒复盘弹窗支持直接运行本机 Codex CLI 生成 Markdown 复盘报告，不需要 OpenAI API Key；同时保留复制 AI 提示词和本地复盘报告下载。
- **移除 Trama 集成**：自动 AI 复盘改为 Codex CLI 侧车方案，避免 Anthropic / DeepSeek / OpenRouter 等 provider 配置混乱。
- **打包兼容修复**：Worker 与指标模块加入 `asarUnpack`，确保安装包环境下后台线程可读取脚本。

## 功能特性

### 主界面

- **RSI 热力图**：以气泡图展示当前周期 RSI 分布，支持市场类型、成交额 TopN、RSI 区间与分组筛选。
- **统计表格**：展示价格、成交额、24h 涨跌幅、15m / 1h / 4h / 1d RSI、评分、量价信号与走势图。
- **搜索与排序**：支持品种搜索、成交额排序、RSI 排序、涨跌幅排序、评分排序。
- **市场情绪分布**：展示超买、强势、中性、弱势、超卖比例。
- **背离标记**：列表与 K 线图中显示看涨 / 看跌背离提示。

### K 线图

- K 线图 + RSI 副图。
- 自动标记背离连线。
- RSI-based MA：支持 SMA / EMA / RMA / WMA。
- RSI Bollinger Bands：支持 BB 倍数配置。
- 支持 15m / 1h / 4h / 1d 周期切换。

### 提醒系统

- 支持 RSI 阈值、价格突破 / 跌破、百分比偏离、24h 涨跌幅、RSI 背离、量价结构信号。
- 支持共振提醒：所选周期全部满足条件时触发特殊提醒；单周期满足时仍触发普通提醒。
- 支持提醒等级、不同等级冷却时间、静音时段。
- 支持批量启用 / 禁用规则。
- 支持右下角桌面弹窗提醒、声音提醒。
- 支持 Telegram Bot 与 Discord Webhook。
- Webhook 默认只发送经过 AI 筛选后的重点提醒，避免普通规则噪音直接推送到外部渠道。
- 提醒记录可按类型和品种过滤，支持 CSV 导出。
- 提醒记录可点击复盘：查看触发价格、当前价格、触发后变化，并可打开 K 线图定位。

### AI 候选池

- 本地规则持续生成当前候选池，适合实时观察市场雷达。
- Codex 筛选会冻结当时的候选快照，返回重点 / 观察 / 忽略 / 风险分类、置信度、原因、风险与下一步观察点。
- 支持后台自动 AI 筛选，可配置冷却时间、启动延迟和每次候选数量。
- 支持单品种 AI 筛选：对某个候选单独运行 Codex，并把结果合并回 AI 结果快照。
- AI 筛选结果可写入提醒记录，并按现有弹窗、声音、Webhook 规则发送。

### Codex 复盘

提醒复盘弹窗提供四种方式：

- **运行 Codex 复盘**：调用本机 Codex CLI，通过当前 ChatGPT / Codex 登录状态生成 Markdown 复盘报告。
- **打开 Codex 目录**：打开本次复盘生成目录。
- **打开 Codex 报告**：打开生成的 `report.md`。
- **复制 AI 提示词**：复制完整 prompt，可粘贴到 Codex / Claude / ChatGPT。
- **下载本地复盘**：不调用 AI，直接根据提醒快照生成本地 Markdown 报告。

Codex 复盘默认使用：

```bash
codex exec --skip-git-repo-check --sandbox read-only
```

报告默认保存到：

```text
C:\Users\<你的用户名>\.rsi-inspection\codex-reviews\
```

### 品种管理

- 币安现货 / 永续合约一键勾选。
- 美股 Ticker 验证后加入观察列表。
- 加密货币与美股列表均支持搜索、全选、取消全选。
- 支持自定义分组，首页和提醒页均可按分组筛选或批量选择。

### 设置

- 刷新间隔、提醒冷却、RSI 周期、超买 / 超卖阈值。
- RSI 均线类型、均线周期、布林带倍数。
- 静音时段、开机自启动、启动最小化。
- 桌面弹窗 / 声音 / Webhook 最低提醒等级。
- Codex CLI 路径检测。
- Telegram / Discord 通知配置与测试。
- 配置导入 / 导出，导出时不会包含 Telegram / Discord 密钥。
- GitHub Releases 更新检查。

## 安装与使用

### 直接安装

从 [Releases](https://github.com/waruiiko/RSI-inspection/releases) 下载最新 `.exe` 安装包，双击安装即可。

### 开发模式

```bash
npm install
cd renderer
npm install
cd ..
npm run dev
```

### 打包

```bash
npm run dist
```

输出目录：

```text
dist/
```

## 数据说明

- 加密货币通过 Binance 公共 API 拉取 OHLCV 数据，无需 API Key。
- 美股通过 Yahoo Finance 非官方接口，仅在美股交易时间内更新。
- 所有 RSI、背离、量价信号均在本地计算。
- K 线数据会缓存在本地，用于减少重复请求和接口临时失败时的回退显示。

## 配置文件位置

设置、提醒规则、自选列表、提醒记录等保存在 Electron `userData` 目录。Windows 下默认位置：

```text
%APPDATA%\market-rsi\
```

Telegram、Discord 等通知密钥只保存在本机用户配置中，不会进入 Git，也不会通过配置导出功能导出。
